# Flipkart Sentiment AI

Streamlit app for analyzing Flipkart product reviews with sentiment labels:
Positive, Neutral, and Negative.

## Run locally

```bash
streamlit run app.py
```

## Files

- `app.py` - Streamlit dashboard
- `sentiment_utils.py` - shared preprocessing and model code
- `Ds_Capstone_p.py` - analysis and evaluation script
- `flipkart.csv` - dataset used by the model

## Streamlit Cloud

Deploy this repository on Streamlit Community Cloud and set the entrypoint to `app.py`.
